What is AV Simulator?

A vehicle simulator for autonomous driving research, built on top of a Carla Simulator.

It provides a realistic environment for developing and testing algorithms related to autonomous vehicles.

It offers a high-fidelity simulation environment with realistic physics, graphics, and sensor models, making it a valuable tool for algorithm evaluation and training.

Built on the Unreal Engine, the AV simulator provides a customizable setting where users can create various scenarios, including different road networks, traffic patterns, and weather conditions.

It supports range of sensors such as lidar, radar, & cameras, enabling developers to test perception algorithms effectively. With its Python API and active community support, CARLA Simulator has become a popular choice for researchers & developers in the field of autonomous vehicles.

Features of AV Simulator

  • High-fidelity simulation

    It provides a highly realistic environment with accurate physics, graphics, and sensor models.

  • Vehicle Integration

    Users can design their own vehicle and simulate it in the simulator.

  • Vehicle Dynamics

    Users can integrate actual vehicle dynamics to simulate accurate physics simulations.

  • Customizable scenarios

    Users can create custom scenarios by defining road networks, traffic patterns, and various weather and lighting conditions.

  • Sensor simulation

    The AV simulator supports various sensors like lidar, radar, and cameras, allowing developers to test perception algorithms.

  • Autonomous Driving API

    It offers an API for controlling vehicles and interacting with the simulator.

  • Integration with ROS

    The CARLA ROS Bridge provides seamless connection between CARLA Simulator and the Robot Operating System (ROS). ROS is a popular framework for developing robotic applications & is widely used in robotics community.
